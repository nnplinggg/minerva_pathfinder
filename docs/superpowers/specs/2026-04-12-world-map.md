# World Map — M4 Design Spec

**Date**: 2026-04-12
**Status**: Brainstorm complete, ready to plan/implement

---

## Goal

Replace (or extend) the current stamp-card world map screen with an interactive, Sprout Lands–styled pixel map where Mi travels between 5 Minerva college nodes and discovers 3 treasure chests along the way.

---

## Visual Style

- **Asset pack**: Sprout Lands – UI Pack – Basic Pack (user-provided)
- **Aesthetic**: Cute, 2D pixel-art, pastel palette
- **Map background**: Assembled from Sprout Lands tileable surface textures (grass, dirt paths, water borders) — no custom illustrated image required
- **Elements on map**: school nodes (5), treasure chests (3), path connectors between nodes, Mi mascot marker

---

## Map Layout

5 school nodes at distinct geographic-ish positions on the canvas (approximate):

| Node | School | Map position |
|------|--------|-------------|
| 1 | San Francisco | top-left |
| 2 | Berlin | top-right |
| 3 | Buenos Aires | bottom-left |
| 4 | Seoul | bottom-right |
| 5 | London (or Hyderabad) | center |

3 treasure chests placed at natural mid-points along the paths between nodes (not overlapping school nodes).

---

## Interaction Model

### School nodes
- **Locked state**: greyed-out pin, no label, not clickable
- **Unlocked state**: colored Sprout Lands building/castle icon, school name label, clickable
- **Unlock trigger**: completing the adventure for that school (existing stamp mechanic)
- **On click**: opens existing school adventure / summary card (no change to current logic)

### Treasure chests
- **Locked state**: closed chest sprite, padlock icon overlay, not clickable
- **Unlocked state**: open chest sprite, glowing/sparkle effect, clickable
- **Unlock thresholds**:
  - Chest 1 → unlocks after discovering **1** school
  - Chest 2 → unlocks after discovering **2** schools
  - Chest 3 → unlocks after discovering **3** schools
- **On click (unlocked)**: opens a modal popup containing:
  - A fun fact about Minerva (text, ~2–3 sentences)
  - A call-to-action link (placeholder URL for now, user will fill in)
  - A close button (Sprout Lands button style)

### Modal popup structure
```
┌─────────────────────────────┐
│  🎁  Fun Fact               │
│                             │
│  [Fun fact text here]       │
│                             │
│  [→ Link label]  [Close]    │
└─────────────────────────────┘
```

## Technical notes

- **No new framework** — vanilla HTML/CSS/JS, consistent with Decision 001
- **Assets**: Sprout Lands sprites are PNG spritesheets; use CSS `background-position` or `<img>` tags with fixed sprite coords
- **Map canvas**: CSS Grid or absolute positioning on a fixed-size `<div>` (e.g. 800×500px), scaled with `transform: scale()` for mobile
- **Animations**: CSS `@keyframes` for chest sparkle and unlock pulse — no JS animation library
- **Modal**: reuse existing modal pattern from current UI if one exists; otherwise a simple `<div class="modal">` with backdrop

---

## Out of scope (this milestone)

- Animated Mi character walking between nodes
- Sound effects
- Persistent state (localStorage) — chests reset on page reload is acceptable for now
- Custom illustrated map image

---

## Acceptance criteria

- [ ] Map renders with 5 school nodes and 3 treasure chests using Sprout Lands assets
- [ ] School nodes lock/unlock correctly based on completed adventures
- [ ] Treasure chests unlock at correct thresholds (1 / 2 / 3 schools discovered)
- [ ] Clicking an unlocked chest opens modal with fun fact + placeholder link
- [ ] Modal closes cleanly; chest stays in "opened" visual state
- [ ] Layout is mobile-responsive (no horizontal scroll on 375px viewport)
- [ ] No regressions to existing adventure or course-matching flows
